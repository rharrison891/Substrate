# Features & Roadmap

Substrate focuses on removing repetitive WPF boilerplate so developers can
spend more time **designing UI** and less time wiring plumbing.

---

## ✅ Current features

- [x] `Notify` — automatic INotifyPropertyChanged
- [x] Dependency Property generator
- [x] Theme generator
- [x] Fluent icon mapping
- [x] Custom font support
- [x] Docs powered by MkDocs

---

## 🛠 In progress

- [ ] Control generator from XAML templates  
- [ ] Navigation + DI (prototype)
- [ ] `[Report]` developer feedback system  
- [ ] Achievement / Easter-egg framework

---

## 🌱 Planned ideas

- [ ] Visual Studio wizard (VSIX)
- [ ] Theme designer UI
- [ ] XAML parser improvements
- [ ] Better validation helpers
- [ ] Converter helpers
